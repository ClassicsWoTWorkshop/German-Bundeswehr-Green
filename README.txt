Thank you for downloading!

To install:
- Drag the included .wotmod file to the mods\<version> folder your World of Tanks install directory (by default, C:\Games\World_of_Tanks\mods\<version>). (Example: C:\Games\World_Of_Tanks\mods\1.3.0.1)

- If you want support for a few remodels (namely my Dakka Improvement Project, Miruku's Leopard 1A1A1 & Prototyp B, and TheFalkonett's RU 251), drag the "vehicles" folder from within the "ADDON - Mod Support" folder into your RES_MODS\<version> folder (Example: C:\Games\World_Of_Tanks\res_mods\1.3.0.1). 

IMPORTANT NOTE - This only includes skins/icons for the following vehicles: 
- SP 1 C, HWK 12, HWK 30, M 41 90, M 41 90 GF, Panzer 58, Pz. 58 Mutz, Indien Pz., Kanonenjagdpanzer, Kanonenjagdpanzer 105, RU 251, Leopard PT A, Leopard 1, and Rheinmetall Panzerwagen (Spz 57 Rh).

Enjoy!
